# Dog-Island-v1.0
3rd Person Survival RGP Platformer game about an explorer searching for his lost dog on a mysterious island.

  | Input | Action |
  | :------------- | :------------- |
  | W A S D | Forward Left Down Right |
  | Space bar | Jump |
  | C | Crouch |
  | Shift | Roll |
  | R | Reload |
  | Left-click |Fire |
  | Right-click |Aim |
  | 1, 2, 3, 4 | Cycle through weapons |
